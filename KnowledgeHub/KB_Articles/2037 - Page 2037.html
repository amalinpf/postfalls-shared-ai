Attachments on emails sent to a SharePoint list could be missing. <\/span>This can especially be an\u00a0issue for attachments on emails sent from Office Outlook for Mac and Outlook Web Access.\u00a0<\/span><\/div>\n
\u00a0<\/span><\/div>\n
Are attachments missing from e-mail messages that are sent to a SharePoint document library?<\/span><\/strong><\/div>\n
If attachments are missing from e-mail messages that are sent to a SharePoint Server 2010 document library, it might be because you associated the document library with an e-mail address. When you do this, Directory Management Service may not add the following two attributes:<\/span><\/div>\n
\u00b7\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 <\/span><\/span>internet Encoding = 1310720<\/span><\/strong><\/div>\n
\u00b7\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0\u00a0 <\/span><\/span>mAPIRecipient = false<\/span><\/strong><\/div>\n
You must use Active Directory Service Interfaces (ADSI) to manually add these two missing attributes.<\/span><\/div>\n\n\n\n\n
\n
Note: <\/span><\/strong><\/div>\n<\/td>\n<\/tr>\n
\n
On servers that are running Windows Server 2008 or Windows Server 2008 R2, ADSI\u00a0Edit is installed when you install the AD\u00a0DS role to make a server a domain controller. You can also install Windows Server 2008 Remote Server Administration Tools (RSAT) on domain member servers or stand-alone servers. For more information, see Installing or Removing the Remote Server Administration Tools Pack8<\/sup><\/span><\/a> (http:\/\/go.microsoft.com\/fwlink\/?LinkId=143345<\/a>).<\/span><\/div>\n<\/td>\n<\/tr>\n<\/tbody>\n<\/table>\n
To add attributes by using ADSI Edit<\/span><\/strong><\/div>\n
    \n
  1. Click Start<\/strong>, and then click Run<\/strong>.<\/span><\/li>\n
  2. In the Run<\/strong> dialog box, type Adsiedit.msc<\/strong>, and then click OK<\/strong>.<\/span><\/li>\n
  3. In the ADSI Edit window, expand ADSI Edit<\/strong>, expand Domain [DomainName]<\/strong>, expand DC=DomainName, DC=com<\/strong>, and then expand CN=Users<\/strong>. <\/span><\/li>\n
  4. Right-click the user name to which you want to add the missing attributes, and then click Properties<\/strong>.<\/span><\/li>\n
  5. In the Properties<\/strong> dialog box, double-click internet Encoding<\/strong> on the Attribute Editor<\/strong> tab.<\/span><\/li>\n
  6. In the Integer Attribute Editor<\/strong> dialog box, type 1310720<\/strong> in the Value<\/strong> box, and then click OK<\/strong>.<\/span><\/li>\n
  7. In the Properties<\/strong> dialog box, double-click mAPIRecipient<\/strong> on the Attribute Editor<\/strong> tab.<\/span><\/li>\n
  8. In the Boolean Attribute Editor<\/strong> dialog box, click False<\/strong>, and then click OK<\/strong> two times.<\/span><\/li>\n<\/ol>\n","protected":false},"excerpt":{"rendered":"
